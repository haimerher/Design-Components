Basic Components

the basic elements have a simple variation to their natural style. You can use it at 100% with the class .form-input

Disabled Elements

Only add the disabled attribute to your elements.

HTML Code
<button class="ds-btn-primary" disabled>Send</button>

Error

Add the class .error at your element to indicate there is something wrong.

HTML Code
<input type="text" class="error"/>

Custom Fields

Adds background image and changes the appearance of the fields form, to indicate at the users what kind of fields is.

Inside Icon

HTML Code
<input type="text" class="inp-user"/>

Other Style

HTML Code
<div class="box-inp">
   <input type="text"/>
   <i class="ion-android-person"></i>
</div>

Switch

Use the switch for simple On/Off options.

HTML Code
<label class="switch">
   <input type="checkbox" id="" class="switch-inp" />
   <span class="switch-label" data-on="Yes" data-off="No"></span>
   <span class="switch-btn"></span>
</label>
