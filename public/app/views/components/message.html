Dialog Message

Presents information messages in this simple dialog box. Put the class .dialog-blue in your content element to use the style.

Information

Aliquam porttitor mauris sit amet orci. Aenean dignissim pellentesque felis.

HTML Code
<div class="dialog-blue">
   <h3>...</h3>
   <p>...</p>
</div>
If you wish to use a different color, you can try with these: .dialog-midnight-blue | .dialog-green | .dialog-green-sea | .dialog-orange | .dialog-red | .dialog-silver | .dialog-violet | .dialog-yellow
