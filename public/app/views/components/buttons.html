Buttons

You can use the style within the tags <a>, <button> and <input>. To apply extra colors at the buttons, you can use the following classes: .ds-btn-violet | .ds-btn-regal-blue | .ds-btn-green-sea

HTML Code
<button class="ds-btn-default">Default</button>
<button class="ds-btn-primary">Primary</button>
<button class="ds-btn-success">Success</button>
<button class="ds-btn-warning">Warning</button>
<button class="ds-btn-danger">Danger</button>

Icons

Do you wants icons on your buttons? Use them alone or with text.

Settings
HTML Code
<button class="ds-btn-primary"><i class="ion-android-settings"></i> Settings</button>
For more icons, consult the library at ionicons.

Flat

Transparent buttons until you make mouseover on them. Put the class .ds-btn-flat and after the color you want.

HTML Code
<button class="ds-btn-flat">Default</button>
<button class="ds-btn-flat blue">Blue</button>
<button class="ds-btn-flat red">Red</button>
<button class="ds-btn-flat orange">Orange</button>
<button class="ds-btn-flat green">Green</button>
<button class="ds-btn-flat violet">Violet</button>
<button class="ds-btn-flat regal-blue">Regal-Blue</button>
<button class="ds-btn-flat green-sea">Green-Sea</button>

Rounded

Put the class .rounded for use a curved style.

HTML Code
<button class="ds-btn-violet rounded">Violet</button>

Raised

Use the class .raised within your element to get raised buttons and make the effect sinking clicking.

HTML Code
<button class="ds-btn-success raised">Success!</button>

Social Icons

For circular buttons, use the class .circle.

HTML Code
<button class="ds-social-tt circle"><i class="ion-social-twitter"></i></button>
<button class="ds-social-fb circle"><i class="ion-social-facebook"></i></button>
<button class="ds-social-gg circle"><i class="ion-social-googleplus"></i></button>
<button class="ds-social-ig circle"><i class="ion-social-instagram"></i></button>
<button class="ds-social-pt circle"><i class="ion-social-pinterest"></i></button>
<button class="ds-social-yt circle"><i class="ion-social-youtube"></i></button>
Think big! Use the other classes of the buttons to get other effects.

Group

You can group many elements in a row. Put the class .btn-group on your parent element.

HTML Code
<div class="btn-group">
   <button class="ds-btn-default"><i class="ion-android-create"></i> Edit</button>
   <button class="ds-btn-default"><i class="ion-android-delete"></i> Delete</button>
   <button class="ds-btn-default"><i class="ion-android-cloud"></i> Download</button>
</div>

Size

To change the size of the buttons, use the classes .btn-sm o .btn-lg

HTML Code
<button class="ds-btn-default btn-sm">Small</button>
<button class="ds-btn-default">Normal</button>
<button class="ds-btn-default btn-lg">Large</button>
If you going to use grouped buttons, is not necessary put the style to each element. Only add the class at its parent element.
