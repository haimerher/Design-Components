Buttons

You can use the style within the tags <a>, <button> and <input>. To apply extra colors at the buttons, you can use the following classes: .violet-btn | .regal-blue-btn | .green-sea-btn

HTML Code
<button class="default-btn">Default</button>
<button class="primary-btn">Primary</button>
<button class="green-btn">Success</button>
<button class="orange-btn">Warning</button>
<button class="red-btn">Danger</button>

Icons

Do you wants icons on your buttons? Use them alone or with text.

Settings
HTML Code
<button class="primary-btn"><i class="ion-android-settings"></i> Settings</button>
For more icons, consult the library at ionicons.

Flat

Transparent buttons until you make mouseover on them. Put the class .flat-btn and after the color you want.

HTML Code
<button class="flat-btn">Default</button>
<button class="flat-btn blue">Blue</button>
<button class="flat-btn red">Red</button>
<button class="flat-btn orange">Orange</button>
<button class="flat-btn green">Green</button>
<button class="flat-btn violet">Violet</button>
<button class="flat-btn regal-blue">Regal-Blue</button>
<button class="flat-btn green-sea">Green-Sea</button>

Rounded

Put the class .rounded for use a curved style.

HTML Code
<button class="violet-btn rounded">Violet</button>

Raised

Use the class .raised within your element to get raised buttons and make the effect sinking clicking.

HTML Code
<button class="green-btn raised">Success!</button>

Social Icons

For circular buttons, use the class .circle.

HTML Code
<button class="social-btn-tt circle"><i class="ion-social-twitter"></i></button>
<button class="social-btn-fb circle"><i class="ion-social-facebook"></i></button>
<button class="social-btn-gg circle"><i class="ion-social-googleplus"></i></button>
<button class="social-btn-ig circle"><i class="ion-social-instagram"></i></button>
<button class="social-btn-pt circle"><i class="ion-social-pinterest"></i></button>
<button class="social-btn-yt circle"><i class="ion-social-youtube"></i></button>
Think big! Use the other classes of the buttons to get other effects.

Group

You can group many elements in a row. Put the class .btn-group on your parent element.

HTML Code
<div class="btn-group">
   <button class="default-btn"><i class="ion-android-create"></i> Edit</button>
   <button class="default-btn"><i class="ion-android-delete"></i> Delete</button>
   <button class="default-btn"><i class="ion-android-cloud"></i> Download</button>
</div>

Size

To change the size of the buttons, use the classes .sm-btn o .lg-btn

HTML Code
<button class="default-btn sm-btn">Small</button>
<button class="default-btn">Normal</button>
<button class="default-btn lg-btn">Large</button>
If you going to use grouped buttons, is not necessary put the style to each element. Only add the class at its parent element.
